Proyecto 3: Deep Learning para Series de Tiempo y Secuencias

Enfoque: Algoritmos y Deep Learning
Tipo de problema: Predicción de series temporales y modelado secuencial.
Dataset sugerido: Air Quality Dataset (UCI) o datos de homicidios dolosos transformados a serie temporal.

Objetivo del proyecto

Comparar arquitecturas profundas para datos secuenciales y analizar cómo las decisiones arquitectónicas afectan la capacidad de representación y generalización.

Checklist de tareas

Estructuración de Datos Temporales
Transformar el dataset en un problema supervisado utilizando ventanas deslizantes (sliding window).

MLP vs LSTM
Diseñar un MLP base y una red LSTM.
Analizar cómo el mecanismo de memoria de la LSTM resuelve limitaciones de las RNN clásicas.

Autoencoders y Embeddings
Implementar un autoencoder para reducción de ruido o generación de embeddings de la serie temporal.

Regularización y Activación
Probar distintas funciones de activación (ReLU, Tanh) y técnicas de regularización (Dropout, Early Stopping).
Evaluar su impacto en overfitting y underfitting.

Visualización del Aprendizaje
Graficar curvas de pérdida de entrenamiento y validación para interpretar el proceso de aprendizaje.

Indicaciones para Colab

Entorno: Python 3 con GPU

Librerías sugeridas: TensorFlow/Keras o PyTorch, matplotlib
